## 🚀 Tech Stack

- **Core**: React 18, TypeScript, Vite 6
- **Styling**: Tailwind CSS v4, Custom Glassmorphism, Google Fonts (*Space Grotesk*, *Plus Jakarta Sans*)
- **Icons**: Lucide React Icons
- **State & Persistence**: React Context API + LocalStorage with JSON backup/export capability
- **Routing & Deployment**: Vite SPA + `vercel.json` for Vercel edge deployment

## 🛠️ Local Development Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/muhammad-talha-portfolio.git
   cd muhammad-talha-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start local development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` in your browser.

4. **Build for production**:
   ```bash
   npm run build
   ```

---

## 🌐 Deploying to Vercel

This project includes a `vercel.json` configuration for single-page routing:
```bash
npx vercel --prod
```
